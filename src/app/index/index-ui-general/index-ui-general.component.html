盒子组件

share-box

名称: share-box

可配置参数: loading、toolCollapse、toolRemove、styleSolid、styleCollapsed、styleTheme

可监听事件：eventCollapse和eventRemove

内容投影：name="body",name="title"

使用方法:

<share-box
  [loading]="true"
  [toolCollapse]="true"
  [toolRemove]="true"
  [styleSolid]="true"
  [styleCollapsed]="false"
  [styleTheme]="4"
  (eventCollapse)="eventCollapse()"
  (eventRemove)="eventRemove()">
<ng-container name="title">此处是标题</ng-container>
<ng-container name="body">此处是内容主体部分</ng-container>
</share-box>

分页

share-page-bar

使用方法

插件名称
share-page-bar
输入型参数:

curPage、totalPage、size

curPage(必传):当前页码 number类型
totalPage(必传):页码综述 number类型
size(选传):分页大小 string类型 不传时默认大小，还有另外一个可传值's'
输出型参数:

changePage

这是一个输出型属性，可以通过这个属性会获得一个参数当前点击的页码
具体使用方法:

不要忘记在主组件中定义change()函数哦

<share-page-bar curPage="1" totalPage="3" (changePage)="change($event)"></share-page-bar>
